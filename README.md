🌿 Greenden – Plant Store Website
📌 Project Overview

Greenden is a responsive multi-page plant store website developed using HTML5 and Tailwind CSS.
This project was created for academic purposes to demonstrate frontend development skills, responsive design principles, and modern CSS utility frameworks.

The website showcases plant products, customer reviews, and a contact form in a clean and minimal user interface.

🚀 Features

✅ Responsive Navigation Bar

✅ Mobile-First Design

✅ Tailwind CSS Utility Styling

✅ Product Listing Page

✅ Customer Review Section

✅ Contact Form

✅ Newsletter Subscription Section

✅ Sticky Navbar

✅ Hover Animations & Effects

🛠 Technologies Used

Tailwind CSS (v4.2.1)
